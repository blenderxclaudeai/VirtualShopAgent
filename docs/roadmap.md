# Roadmap

## Phase 1 — MVP wiring (now)
- Extension button + overlay flow
- Supabase `tryon-request` edge function
- Stable request/response contract (`resultImageUrl`)

## Phase 2 — Product quality
- Better product-image extraction per retailer template
- Improve generated image quality and consistency
- Add loading/retry/error UX polish

## Phase 3 — User profile
- Dashboard auth
- Upload body/face/room images
- Personal context-aware generation

## Phase 4 — Production hardening
- Observability, tracing, latency targets
- Abuse prevention / moderation
- A/B testing on conversion lift and return-rate impact

> Cashback/wallet remains out of scope for current MVP.
